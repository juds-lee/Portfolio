My Skills

My Projects

Who's that Pokemon

Who's that Pokemon is a guessing game reminiscent to the one that played on TV during my childhood. Users must guess the correct pokemon based on their shadows and submit their answer. With every correct guess, the corresponding Pokemon will get added to their Pokedex.

This game was creating using React, HTML5, and CSS.

Quiztionary

Quiztionary is a multiple choice dictionary game where players must match the correct word to the definition. It relies on two different API's to generate random words (Datamuse API) and to procure a matching definition (Merriam-Webster Dictionary API). Made in collaboration with Stephen Korzenstein.

This game was creating using Vanilla.Js, HTML5, and Sass.

Under The Sea

Under the Sea is a photoshop design conversion created with mobile users in mind. It is fully responsive with a functioning image coursel, comments section, and menu.

This app was created using Vanilla.Js HTML5, and Sass.
